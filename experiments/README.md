# Experiments

Experiments are grouped by broad systems topic:

- `memory/`
- `processes/`
- `filesystems/`
- `networking/`
- `concurrency/`
- `cpu/`

Use `experiments/<topic>/<experiment-name>/` for each new experiment.

Keep each experiment focused on one concept. If an idea needs multiple concepts,
split it into multiple experiments and link them from the READMEs.
