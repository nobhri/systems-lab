# Session retrospectives

Use this directory for short notes at the end of learning sessions when the
notes are likely to help future work.

Retrospectives are optional. They are useful for:

- Error patterns and fixes
- Decisions about experiment structure
- Observations that connect multiple experiments
- Follow-up questions for a later session

Name files as `YYYY-MM-DD-NN-topic.md`, where `NN` is a two-digit sequence
number for that date.

Example:

```text
2026-07-10-01-memory-layout.md
```
